Hexo基本操作全解析

Hexo是一款轻量级的静态博客框架，基于Node.js构建，上手简单且功能强大。本文不仅会讲解Hexo的核心操作命令，还会结合Markdown语法示例，让你一站式掌握Hexo帖子的编写规范。

一、Hexo核心命令（必掌握）

以下是Hexo日常使用中最常用的命令，所有命令均需在Hexo博客根目录执行。

1. 项目初始化（首次搭建）

如果是第一次搭建Hexo博客，先执行初始化命令：

# 创建并初始化Hexo博客目录
hexo init my-hexo-blog
# 进入博客目录
cd my-hexo-blog
# 安装依赖
npm install
