Hexo项目创建

官方文档地址

Hexo是一款轻量级的静态博客框架，基于Node.js构建，上手简单且功能强大。

Hexo搭建流程

以下步骤为已安装Node.js和Git环境开始，下载Hexo框架到部署第一个博客的流程，如果你还没有上述两个环境可先移步其他教程

下载Hexo框架(用于创建Hexo项目)

如果是第一次搭建Hexo博客，先执行初始化命令：
npm install -g hexo-cli

创建Hexo项目

  1. 先进入你想创建项目的文件夹
    cd: 用于移动目录位置
    这里我在D 盘的project文件夹里创建并进入
  2. hexo初始化博客
    hexo init <folder>
    其中<floder>为你要创建的博客项目名
    这里我创建名为myBlog的博客
    D:\project\exampleBlog>hexo init myBlog
    显示以下内容即为创建成功
  3. 进入你创建的博客项目文件夹并安装依赖
    进入文件夹依旧使用cd：
    D:\project\exampleBlog>cd myBlog
    安装依赖
    npm install
    弹出即为成功

3.运行项目

在你所创建的文件夹使用Hexo 命令

  1. 生成我们博客的网页文件
    Hexo generate 或者Hexo g(缩写)
  2. 运行博客服务
    Hexo server 或者 Hexo s(缩写)
  3. 看看我们刚生的崽
    打开 http://localhost:4000/ 就能看到默认的初始博客了
